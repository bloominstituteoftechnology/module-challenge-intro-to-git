1. What is Semantic HTML? 
        Used to convey meaning beyond presentational value. For example, a h1 is used to emphasize an important header.
    2. What is HTML used for? 
        HTML is used to create electronic documents.
    3. What is an attribute and where do we put it? 
        Attributes are used to modify functionality or provide functionality to elements.
        We put it inside the element's opening tag.
    4. What is the h1 tag used for? How many times should I use it on a page?
        An h1 tag is used to emphasize importance. It should only be used once.
    5. Name two tags that have required attributes
        The anchor tag and the image tag (<a>, <img>)
    6. What do we put in the head of our HTML document? 
        Meta data, or info about our data, site.
    7. What is an id? 
        A unique identifier.
    8. What elements can I add an id to? 
        Any.
    9. How many times can I use the same id on a page? 
        Once.
    10. What is a class? 
        A class is an attribute that can be used to identify multiple elements.
    11. What elements can I add a class to? 
        All.
    12. How many times can I use the same class on a page? 
        As many as necessary.
    13. How do I get my link to open in a new tab?
        <a target='_blank'>
    14. What is the alt attribute used for? 
        To specify alternative text incase the image can't be rendered or a screen reader is being used.
    15. How do I reference an id?
        by using a # sign.
    16. What is the difference between a section and a div
        A section is used to define related elements, a div has no associated semantics.
    17. What is CSS used for? 
        To style our page or HTML.
    18. How to we select an element? Example - every h2 on the page

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
        An id can only be used uniquely. A class can be assigned to multiple elements. We would use a class
        to style multiple elements and an id to style something unique.
    20. How do we select classes in CSS?
        By putting a . in front of the class name.

         A parent child selector takes all elemnts that are direct child of the parents.
    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?
        A class that is added to specify a state of an element. Example hover.
