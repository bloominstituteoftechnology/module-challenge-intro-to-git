1.What is Semantic HTML? 
    HTML that introduces meaning to the page vs. just presentation
2. What is HTML used for? 
    to create electronic documents to be displayed on the web
3. What is an attribute and where do we put it? 
    HTML attributes provide additional information: we put them in the opening tag
4. What is the h1 tag used for? How many times should I use it on a page?
    Most important thing on the page, theoretically just once
5. Name two tags that have required attributes
    <a> (href), <img> (src)
6. What do we put in the head of our HTML document? 
    Data about the webpage: title, style sheet, etc.
7. What is an id? 
    An attribute that acts as a uniqu identifier 
8. What elements can I add an id to?
    most, but some cannot: meta, param, script, etc.
9. How many times can I use the same id on a page? 
    once
10. What is a class? 
    An attribute that specifies class names for HTML elements 
11. What elements can I add a class to? 
    Any of them
12. How many times can I use the same class on a page?
    multiple
13. How do I get my link to open in a new tab?
    target ="_blank"
14. What is the alt attribute used for?
    typically for images or something needing an alternative description
15. How do I reference an id?
    #'id'
16. What is the difference between a section and a div
    section is semantic, where as a div is not
17. What is CSS used for? 
    Styling a webpage: "adding the paint/decor to a house"
18. How to we select an element? Example - every h2 on the page

19. What is the difference between a class and an id? - Give me an example of when I might use each one
    Id is to identify only one element, whereas a class can identify more than one

21. How do we select a p element with a single class of “human””?

22. What is a parent child selector? When would this be useful?
    It matches elements matched by a second selector that are the direct children of elements matched by the first
    this could be usefull if for example you wanted to specify only anchor tags that contained img's
23. How do you select all links within a div with the class of sidebar?
    .sidebar 
24. What is a pseudo selector?
    Keyword added to a selctor to specify a specific state i.e. :hover
